红米刷机安装第三方ROM

话说Android手机就是用来折腾的，而且还是小米，不刷刷机都对不起自己，呵呵。

0 设备

  • Windows 8.1系统台式机
  • 红米联通（WCDMA）版

1 root

由于官方recovery只能刷官方的rom，所以要刷机安装第三方rom，必须先刷第三方recovery，而这又要求首先root。

miui红米论坛上root的方法一堆，包括电脑端和手机端都有，详细了解。

先试了手机端，百度root、root大师均不行，root大师提示权限不够。

再试电脑端，mi-tools不行，最后使用root大师成功（PS原来的一个移动版红米同样也只有root大师电脑板能够搞定）。

2 刷第三方recovery

这里选择了cwm的第三方recovery

使用cwm自带的手机连接电脑执行刷入红米CWMrecovery.bat无法成功

只能将其中的Cofface_hongmi_recovery_v1.0.img拷到sd卡中，

安装移动叔叔客户端，通过移动叔叔刷recovery成功。

PS：刷完第三方不要重启进入官方系统，否则recovery会被恢复成官方recovery。

3 SDCard无法挂载问题

一般来说进入第三方recovery后，直接选择SD卡中的第三方rom文件即可，但是此次却出现在recovery中无法挂载SD卡（can‘t mount SDcard）的情况，遇到这种情况可使用panasonic SDcard formatter重新格式化SD即可。

4 rom文件修改慎用记事本

由于我打算刷开启了miui系统中自带root选项的准官方rom，链接一，链接二。

于是需要对官方rom进行编辑修改，修改这些文本文件的过程中慎用系统自带的记事本，不知道什么时候就给你加个空格或换行什么的，导致rom错误无法使用。

推荐使用vim或者ultraedit等文本编辑软件。
