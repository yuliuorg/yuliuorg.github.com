让asp.net startkit使用SQL server 2005

在MSDN上面下了一个aspnet 2.0的startkit

里面使用的都是sql server 2005 express的数据库

但是我没有安装它。只安装了sql server 2005

直接运行会出现错误

关于这个问题郁闷了半天。

后来发现其实只要在web.config修改下面的

connectionString="Data Source=.\SQLExpress;Integrated Security=True;User Instance=True;AttachDBFilename=|DataDirectory|CommerceDB.mdf" providerName="System.Data.SqlClient" />

为

connectionString="Data Source=sqlserver2005数据库实例名;AttachDbFilename=|DataDirectory|CommerceDB.mdf;User ID=sa;Password=******" />

就可以了
