喜欢的软件

日常工作生活常用软件的列表，排名不分先后，但这个东西随时间的变化太快，不保证及时更新，欢迎拍砖。

  1. APPz

    • windows（没有理由不排在第一位）
    • office （你已经发现我是一个微软控了吧）

  2. 系统工具

    • vim （嗯，我是第二类人，主要是*nux的cli下基本都自带vi，而且emacs的门槛太高，最近好像sublime text 2的好评如潮，有空再试试）
    • firefox / chrome （不争了，两个一起用）
    • total command （据说有不少人是因为tc才留在windows的）
    • everything （快速搜索，这个功能难道不是系统应该直接集成的吗？）
    • picpick （还能有比这个更好的截图软件吗）

  3. 工程小软件

    • ConvertAll （单位换算必备，使用python实现，单位组合是其最值得推荐之处）

  4. 程序设计

    • LINQPad （使用LINQ与数据库进行交互和管理的工具，现在不经常用了）
